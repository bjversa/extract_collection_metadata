# Extract OpenSea Collection Data

Como rodar:

1. Execute
```bash
npm i
```

2. Crie um arquivo chamado `.env` e insira nele `API_KEY=<sua api key da opensea>`

3. Execute
```bash
node index.js
```

4. Crie uma pasta chamada `images` dentro da pasta desse projeto e execute:
```bash
node download.js
```
